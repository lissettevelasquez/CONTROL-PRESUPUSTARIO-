Este proyecto muestra un ejemplo de control presupuestario utilizando Python, Excel y Power BI.
El objetivo es ilustrar cómo se puede analizar el cumplimiento de presupuestos, comparar gastos reales versus proyectados y generar visualizaciones dinámicas para la toma de decisiones.

Objetivos del Proyecto:
Monitorear la ejecución presupuestaria por área o centro de costo.
Detectar desviaciones significativas entre presupuesto y gasto real.
Presentar reportes visuales que faciliten el análisis financiero mensual.
Automatizar parte del flujo de análisis con Python.

Herramientas Utilizadas:
Python : para limpieza, transformación y análisis de datos.
Excel: almacenamiento de datos simulados y cálculos base.
Power BI: visualización interactiva de resultados.
GitHub: control de versiones y exposición del proyecto.

Flujo de Trabajo
Carga de datos simulados (presupuesto y gasto real).
Limpieza y estandarización de formatos.
Cálculo de desviaciones (porcentaje y monto).
Análisis por área o cuenta contable.
Creación de dashboard con visualizaciones clave.

El reporte final muestra:
Cumplimiento presupuestario mensual.
Áreas con sobreconsumo.
Tendencias mensuales y anuales de gasto.
Gráficos comparativos (barras, líneas y KPI).
